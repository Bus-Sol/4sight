Sales Proposal

The proposal management module provides a more efficient way to create, store, and manage your proposal creation & submission process with better insight into the sales pipeline and performance. Most sales executives spend hours and hours drafting proposals by curating content & images, outlining the scope, creating timelines, and inputting estimates, etc. — only to do it all over again for the next prospect.


With our module, we eliminate this and allow you to easily create a sales proposal within the Odoo sales module. The modules allow you to set up a default proposal template which you can again customize if needed on a case-to-case basis. With proposal management, you can create multiple pages such as executive summary, introduction, solution description, etc. and you can only choose the pages you need per proposal. Also, all the proposals will remain inside the Odoo sales module for future reference and revisions. With this mode, you no more need to work on external word or excel files to create and submit your proposals.

Setup & Customize Template

Go to Sale --> Configurations --> Proposal Management --> Pages

From here, you can design and add content to the pages for your proposal templates. You can easily create a page such as Executive summary, About us, Our Team, etc. for each section which you can reuse for each proposal


Creating Pages

Here you can design each page with relevant content with styling such as header, body, etc., and add images if needed



You can design the pages with names and contents. By typing the “/” for choosing the variety of commands for managing the basic blocks, medias, etc.

Once the sales quotation is created on the system, then you can trigger directly in to the proposal screen by simply clicking on the smart button


Creating Sales Proposal

Here you can create and edit the proposal and choose only the pages you would like to include


We can add as many pages as we want in this proposal screen and mix and match to add only the required pages and save the proposal, then the proposal reference number will be auto-generated.


You can use the check box “Include Quotation” to attach the quotation directly from the sales module. You can also adjust the proposal page positions by the simple drag and drop option.
Once done you have the option to send by email or print it as a pdf



We can take the printout of the proposal in PDF format.
